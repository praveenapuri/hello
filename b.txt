Build cmd: mvn clean install -Dgit.commit.id=<+codebase.commitSha>

application.properties:   app.git.commit=${git.commit.id:unknown}

You should add this in a Spring-managed class where you already handle UI or configuration:
@Value("${app.git.commit}")
private String commitId;

Short Commit ID(if needed):
String shortCommitId = commitId != null ? commitId.substring(0, 7) : "unknown";
